# Desafio_Tecnico_Navian
A partir de la MRI usando UnityVolumeRendering y opcionalmente sus objetos 3D, diseñá e implementá una escena interactiva de temática libre relacionada con visualización médica o exploración 3D/XR. La propuesta puede ser simple, pero debe mostrar criterio técnico, claridad de interacción y atención al detalle.
